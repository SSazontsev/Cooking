## 🚀 Быстрый старт (CLI Mode)

Если вы не хотите открывать тяжелую Android Studio, вот кратчайший путь к готовому APK прямо из терминала.

### 1. Установка инструментов сборки (только для macOS)
Если у вас еще нет Android SDK, проще всего поставить его через Homebrew:

```bash
brew install --cask android-platform-tools
brew install --cask android-commandlinetools
```

### 2. Настройка среды
Пропишите путь к SDK в файле `local.properties` (создайте его в корне проекта, если его нет):

```properties
sdk.dir=/Users/YOUR_USER/Library/Android/sdk
```
*(Замените `YOUR_USER` на свое имя пользователя в системе)*.

### 3. Принятие лицензий
Перед первой сборкой обязательно примите условия использования Google:

```bash
yes | sdkmanager --licenses
```

---

## 📦 Сборка APK

Запустите процесс компиляции и упаковки приложения одной командой:

```bash
./gradlew assembleDebug
```

Система сама скачает зависимости и соберет проект. Это может занять несколько минут.

### 🏁 Результат сборки
Когда в терминале появится надпись **`BUILD SUCCESSFUL`**, ваш готовый файл будет ждать вас здесь:

📂 **Путь к файлу:**
`app/build/outputs/apk/debug/app-debug.apk`
